## 问题修复记录

### 问题 1: Prompt 模板 JSON 转义错误

**症状**: 投资模块 AI 分析失败，错误 `KeyError: '\n  "summary"'`

**根本原因**: Prompt 模板中的 JSON 示例包含未转义的花括号 `{}`，被 Python `str.format()` 误认为占位符

**修复方案**:
1. `prompts/investment_step1_article.txt` - 转义 JSON 示例中的花括号
2. `wechat/prompts/wechat_step2_aggregate.txt` - 转义两个 JSON 示例中的花括号

**验证**: ✅ 投资模块 AI 分析正常工作

### 问题 2: 播客模块使用测试数据

**症状**: 播客邮件只包含硬编码的测试数据，内容不完整

**根本原因**: 测试脚本使用模拟数据而非真实 AI 分析

**修复方案**:
1. 修改 `agents/prerelease_e2e_test.py` 中的 `test_podcast()` 方法
2. 导入并使用 `PodcastAnalyzer` 进行真实 AI 分析
3. 使用节目简介 (episode.summary) 作为分析输入
4. 修正属性名：`episode.description` → `episode.summary`

**验证**: ✅ 播客模块生成完整的 AI 分析（4560 字符）
