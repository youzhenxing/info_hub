```python
def _format_news_time(self, published: str, now: datetime) -> str:
    """格式化新闻时间为相对时间或绝对时间"""
    time_diff = (now - pub_time).total_seconds()

    if time_diff < 86400:  # 24小时内
        hours = int(time_diff / 3600)
        if hours == 0:
            return "刚刚"
        elif hours == 1:
            return "1小时前"
        else:
            return f"{hours}小时前"
    else:
        return pub_time.strftime("%Y-%m-%d")
```

## 🎯 功能特性

### 1. 智能时间过滤

**A股/港股新闻**:
- 今天00:00起 → 保留
- 昨天或更早 → 过滤

**美股新闻**:
- 今天00:00起 → 保留
- 昨天21:00起 → 保留
- 昨天21:00前 → 过滤

**通用规则**:
- 无时间信息 → 保留（无法判断）

### 2. 相对时间显示

| 时间范围 | 显示格式 | 示例 |
|---------|---------|------|
| 0-60秒 | "刚刚" | 刚发布 |
| 1小时 | "1小时前" | 1小时前 |
| 2-23小时 | "X小时前" | 3小时前 |
| ≥24小时 | "YYYY-MM-DD" | 2026-02-06 |

### 3. 用户体验优化

**优势**:
- ✅ 更直观："3小时前"比"15:31"更容易理解
- ✅ 更新鲜：突出最新新闻的重要性
- ✅ 更精确：立即识别新闻时效性

**适用场景**:
- 🔥 热点新闻：显示"刚刚"
- 今日新闻：显示"X小时前"
- 历史新闻：显示完整日期
