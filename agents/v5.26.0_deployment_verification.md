## 部署问题与解决

### 问题 1：版本切换时容器冲突

**错误信息**：
```
Error response from daemon: Conflict. The container name "/trendradar-mcp-prod" is already in use
```

**解决方案**：
```bash
docker stop trendradar-mcp-prod trendradar-prod
docker rm trendradar-mcp-prod trendradar-prod
yes "y" | trend update v5.26.0
```

**结果**：✅ 容器成功重建
