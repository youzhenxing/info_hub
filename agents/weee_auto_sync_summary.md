**更新时间**: 2026-02-07 23:45

## ✅ 核心结论：**不需要每次手动点击！**

### 正常情况：完全自动 ✅

Wewe-RSS 配置了**每2小时自动同步**：
```yaml
CRON_EXPRESSION=0 */2 * * *
```

**自动工作流程**:
```
00:00 → 自动同步
02:00 → 自动同步
04:00 → 自动同步
...
22:00 → 自动同步

23:00 → 微信服务自动采集 + 推送
```

**你的日常工作**: 无任何操作 ✅

---

## ⚠️ 什么时候需要手动操作？

### 只有这种情况：**微信读书账号失效**

**发生频率**: 偶尔（可能几周或几个月一次）

**如何判断**:
1. 收到告警邮件（如果设置了监控）
2. 微信服务采集不到最新文章
3. Wewe-RSS 日志显示："暂无可用读书账号"

**解决方法**（只需5分钟）:
```bash
1. 访问 http://localhost:4000
2. 点击"账号管理"
3. 扫描二维码重新登录
4. 点击"立即同步"（可选，等2分钟也行）
```

**完成后**: 自动恢复，无需其他操作

## 🛡️ 如何提前知道账号失效？

### 方案1: 添加自动监控（推荐）⭐

我已经创建了监控脚本：
- 文件位置: `agents/monitor_wewe_account.py`
- 功能: 自动检测账号状态，失效时发送邮件告警

**添加到定时任务**:
```bash
# 编辑 crontab
crontab -e

# 添加这一行（每天凌晨3点检查）
0 3 * * * cd /home/zxy/Documents/code/TrendRadar && python3 agents/monitor_wewe_account.py >> logs/wewe_monitor.log 2>&1
```

**效果**:
- ✅ 每天自动检查账号状态
- ✅ 失效时发送邮件到 {{EMAIL_ADDRESS}}
- ✅ 提供详细处理步骤

### 方案2: 定期手动检查（可选）

```bash
# 运行监控脚本
python3 agents/monitor_wewe_account.py
```
