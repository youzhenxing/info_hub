## 🔍 常用命令

### 查看容器状态
```bash
docker ps | grep trendradar
```

### 查看主服务日志
```bash
docker logs trendradar -f
```

### 查看 MCP 服务日志
```bash
docker logs trendradar-mcp -f
```

### 重启服务
```bash
docker restart trendradar
docker restart trendradar-mcp
```

### 停止服务
```bash
cd docker
docker compose -f docker-compose-build.yml down
```

### 重新启动服务
```bash
cd docker
docker compose -f docker-compose-build.yml up -d
```

## ⏰ 定时任务

服务已配置为每 30 分钟自动抓取一次（`*/30 * * * *`）

如需修改频率，编辑 `docker-compose-build.yml` 中的 `CRON_SCHEDULE` 环境变量。

## 🌐 Web 服务器（可选）

如需启用 Web 服务器，修改 `docker-compose-build.yml`：

```yaml
environment:
  - ENABLE_WEBSERVER=true
  - WEBSERVER_PORT=8080
```

然后重启容器：
```bash
docker restart trendradar
```

访问地址: http://127.0.0.1:8080
