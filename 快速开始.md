## ⚡ 快速命令

### 使用 trend 命令（推荐） ⭐

```bash
# 查看服务状态
./trend info

# 手动触发一次抓取和推送
./trend trigger

# 查看实时日志
./trend logs

# 重启服务
./trend restart

# 停止服务
./trend stop

# 编辑配置
./trend config

# 打开最新报告
./trend report

# 查看帮助
./trend help
```

### 设置全局命令（可选）

添加到 `~/.bashrc` 或 `~/.zshrc`：
```bash
alias trend='/home/zxy/Documents/code/TrendRadar/trend'
```

之后可以在任何目录使用：
```bash
trend info
```
